# tcltotux
`tcltotux.tcl` is a tcl script that converts Tux Racer 0.61 `course.tcl` files to Extreme Tux Racer `course.dim`.

## Running/Installing

### Prerequisites
- `Tclsh` installed to `/usr/bin/`.

### Running
`./tcltotux.tcl /path/to/course.tcl`
